# 2027학년도 수시지원 대시보드

`2027학년도 수시지원의 모든 것 V6.12.xlsx` 자료를 가공한 **학생·학부모용 수시 분석 대시보드**입니다.
전국 164개 대학 · 26,411개 전형을 계열 카테고리(18개 대분류 + 메디컬·공학 세부 11개)로 나눠, **2026 대비 변화**와 그 변화가 **입결에 미칠 영향**, **3개년(2024→2026) 입결·경쟁률 추이**를 한눈에 보여줍니다.

## 여는 방법
1. (가장 간단) `index.html` 파일을 더블클릭 → 브라우저에서 바로 열립니다. 인터넷 연결이 필요 없습니다.
2. 일부 브라우저에서 `data.js`가 차단되면, 이 폴더에서 로컬 서버로 여세요:
   ```
   cd dashboard
   python3 -m http.server 8000
   ```
   그 후 브라우저에서 `http://localhost:8000` 접속.

## 주요 기능
- **계열 카테고리**: 메디컬(└ 의예·치의예·한의예·수의예·약학), 간호·보건, 공학(└ 컴퓨터·SW·AI / 전기·전자·반도체 / 기계·자동차·항공 / 화공·소재·바이오 / 건설·건축·환경 / 산업·기타), 자연, 상경, 어문, 문사철, 비상경, 사회과학, 통계, 반도체, 반도체 계약, 계약학과, 군 계약, 사범, 교대, IST, 자유전공(무전공). 메디컬·공학은 세부 탭으로 펼쳐 비교 가능.
- **⭐ 올해 입시 유불리 예상 (핵심)**: **2026 vs 2025 입결·경쟁률·추합 추이**와 **2027 모집인원·수능최저 변화**를 종합해 전형마다 `유리 / 불리 / 신설 / 중립`을 자동 추정. 표·상세·비교 어디서나 2025→2026 지표 변화와 그 해석을 함께 표시.
  - 입결 하락세·경쟁률 하락·증원·수능최저 강화/신설 → **유리** 신호
  - 입결 상승세·경쟁률 상승·감원·수능최저 완화/폐지 → **불리** 신호
- **🗂️ 지원카드(즐겨찾기) 9장**: 지원희망 1~6순위 + 도전 1~3순위. 표·상세의 ☆ 버튼으로 담고, 드로어에서 ▲▼로 순위 변경·✕ 제거. 브라우저에 자동 저장.
- **대학군 비교 차트** + **3개년 추이 그래프**.
- **전형 목록 표**: 입결·경쟁률 미니 추이(스파크라인) + 전년비 배지 + 유불리, 정렬·검색·필터·페이지.
- **비교함**: 여러 전형을 최대 6개까지 나란히(2025→2026 지표 포함) 비교.
- 다크 모드 · 모바일 반응형 지원.

## 파일 구성
| 파일 | 설명 |
|------|------|
| `index.html`, `styles.css`, `app.js` | 대시보드 (의존성 없음, 순수 바닐라 JS) |
| `data.js` | 가공된 데이터 (딕셔너리 인코딩) |
| `build_data.py` | xlsx → data.js 재생성 스크립트 (`python3 build_data.py`) |
| `audit_categories.json` | 카테고리 분류 결과 점검용 참고 자료 |

## 주의
'올해 입시 유불리 예상'과 '수능최저 변화(완화/강화)'는 공개 데이터 기반 **자동 분석 추정치**로 참고용입니다.
실제 지원 전 반드시 각 대학의 최신 모집요강을 확인하세요.
